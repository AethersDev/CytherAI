PRIVACY PROTOCOL

Privacy Policy

Last Updated: January 12, 2026

Philosophy

CytherAI is built on principles of sovereignty and verification. We apply these same principles to privacy: zero tracking, minimal collection, maximum transparency.

What We DON'T Collect

  • No Analytics: No Google Analytics, Mixpanel, Heap, or similar
  • No Tracking Pixels: No Facebook Pixel, LinkedIn Insight, etc.
  • No Third-Party Scripts: No CDNs, external fonts, or ad networks
  • No Cookies: This website uses zero cookies (verify in DevTools)
  • No Fingerprinting: No canvas fingerprinting or browser tracking
  • No Session Recording: No FullStory, Hotjar, or similar
Verification:

Our homepage includes a live "Sealed Artifact Verification" panel that monitors and displays external requests in real-time. You can verify zero tracking yourself.

What We DO Collect

Server Logs (Minimal)

Standard web server logs for operational purposes:

  • IP address (for security and abuse prevention)
  • Requested URL (for error diagnosis)
  • HTTP status code (for uptime monitoring)
  • User agent string (for compatibility testing)

Retention: 30 days, then deleted. Logs are not shared with third parties.

Contact Form Data

When you submit the contact form:

  • Name, email, organization (you provide)
  • Message content (you provide)
  • Submission timestamp

Purpose: Response to your inquiry only. We don't add you to mailing lists without explicit consent.

Retention: As long as necessary for business relationship, then deleted upon request.

NDA & Commercial Agreements

For customers under NDA or commercial license:

  • Company information (as provided in agreement)
  • Technical contact details
  • Deployment metadata (if customer opts in for support)

Purpose: Legal compliance, support delivery, security notifications.

How We Use Data

  • Respond to your inquiries
  • Fulfill contractual obligations (for customers)
  • Security monitoring and abuse prevention
  • Legal compliance (when required by law)

We never: Sell data, share with advertisers, use for profiling, or train AI models on customer data.

Third-Party Services

Currently: None. This website is self-hosted with zero third-party scripts.

Future: If we ever integrate third-party services (e.g., email provider, payment processor), we will:

  • Update this policy with clear disclosure
  • Choose only privacy-respecting vendors
  • Limit data sharing to minimum necessary
  • Maintain data processing agreements (DPAs)

Your Rights

You have the right to:

  • Access: Request a copy of data we hold about you
  • Correction: Update incorrect information
  • Deletion: Request complete removal of your data
  • Portability: Receive data in machine-readable format
  • Objection: Object to processing (we'll comply unless legal obligation)

To exercise these rights, email: privacy@cytherai.com

Data Security

We protect data using industry-standard practices:

  • TLS encryption for all web traffic
  • Encrypted storage for customer data
  • Access controls and least-privilege principles
  • Regular security audits (internal and external)

Children's Privacy

CytherAI is not intended for users under 18. We do not knowingly collect data from children. If we discover we've collected such data, we'll delete it immediately.

Changes to This Policy

We may update this policy. Material changes will be announced via:

  • Notice on this website (30 days prior)
  • Email to active customers

Continued use after notice constitutes acceptance.

Questions?
